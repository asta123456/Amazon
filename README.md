💻 Amazon Laptop Price Dataset Analysis
📌 Project Overview

This project analyzes the Amazon Laptop Price Dataset to understand laptop pricing trends based on different specifications and brands available on Amazon India.

The goal is to explore how features like RAM, storage, processor, and brand influence laptop prices.

📂 Dataset Information

File Name: amazon_laptop_price_dataset.csv

The dataset includes the following features:

Laptop Name

Brand

Price

RAM

Storage

Processor

Ratings (if available)

🎯 Objectives

Analyze laptop price distribution

Compare prices across different brands

Identify budget, mid-range, and premium laptops

Study the relationship between specifications and price

Find average price per brand

🛠 Technologies Used

Python

Pandas

NumPy

Matplotlib

📊 Analysis Performed

Data cleaning

Handling missing values

Price categorization (Budget / Mid-Range / Premium)

Brand-wise price comparison

Basic statistical analysis (mean, min, max price)

Visualization of price distribution
